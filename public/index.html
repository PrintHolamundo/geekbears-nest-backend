Check documentation at http://167.99.207.48:4000/api/docs

Required Endpoints

  • /signup - Register a new user in the database with an email and password
  • /login - User login with email and password
  • /encode - Encode a URL into a shortened URL. This URL should be private and only accessible once the user has logged in.
  • /decode - Decode a shortened URL to its original URL. This URL should be private and only accessible once the user has logged in.
